## О проекте

Social Media API — это RESTful backend-приложение, разработанное на Java с использованием Spring Boot.

Цель проекта — реализовать социальную платформу, где пользователи могут взаимодействовать друг с другом через публикации, подписки, дружбу и ленту активности.

## Возможности проекта:

- Регистрация и аутентификация пользователей (JWT, защита паролей)
- Создание, редактирование, удаление и просмотр постов с возможностью прикрепления изображений
- Система подписок и друзей (заявки в друзья, принятие и отклонение)
- Лента активности пользователя с постами от подписок
- Пагинация и сортировка постов по времени создания
- Обработка ошибок и валидация входных данных
- Документация API через Swagger / OpenAPI

## Технологии и инструменты

- Язык программирования: Java 17
- Фреймворк: Spring Boot
- База данных: PostgreSQL
- Аутентификация и авторизация: Spring Security + JWT
- Документация API: Swagger / OpenAPI
